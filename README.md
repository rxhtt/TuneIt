TuneIT is a supercharged, high-performance music application that bridges the gap between your local library and the vast world of YouTube Music.

## 🚀 Key Features

TuneIT is built for audiophiles who demand both local control and global access, featuring a sleek, futuristic UI and a high-performance audio engine.

- **Dual-Core Playback**: Seamlessly switch between local media (MP3, OGG, FLAC) and YouTube Music.
- **Advanced Metadata Extraction**: Uses a custom-built TagLib integration to ensure your local tags are perfectly parsed.
- **Dynamic Premium Theming**: A beautifully crafted UI using Material 3 and custom high-contrast modes.
- **No Advertisements**: Enjoy uninterrupted background playback without any YouTube Ads.
- **Synchronized Lyrics**: Support for synchronized and Karaoke-style lyrics (LRC, TTML).
- **High Fidelity Audio**: Audio normalization, pitch/tempo adjustment, and gapless playback support.
- **Cross-Platform Harmony**: Account synchronization across your devices.
